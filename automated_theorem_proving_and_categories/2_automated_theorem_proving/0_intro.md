# Proving a Theorem 

A machine can prove a theorem. Sometimes you need a machine to prove a theorem (enumerating large number of scenarios). Other times, a machine makes proving a theorem easier (using LLMs to think about the problem differently). I am interested here in looking at how we can set up a programming language, an environment, and then a learning system to prove theorems autonomously.
